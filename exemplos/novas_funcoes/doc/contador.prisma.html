  1. contador

contador (biblioteca base Prisma)

Retorna uma função de fecho (closure) com um valor incrementado ou decrementado a cada chamada.

Exemplo:

📋
 ex1.prisma
imprima('---------- Início padrão: 1, incr padrão: 1 -----');
local cont = contador();//padrão inicia por 1, o incremento é 1;
imprima(cont());
imprima(cont());
imprima(cont());

Funções:

Voltar ao início

funcao contador(inic, incr)

Cria uma função contadora que incrementa ou decrementa seu valor inicial conforme for executada.

Parâmetros:

inic(numero) - é o número inicial, caso seja omitido o valor padrão é 1;
incr(numero) - incremento ou decremento é o número a ser somado a cada execução da função.

Retornos:

(funcao) - o retorno é uma função contadora, basta executá-la e o retorno será um novo valor incrementado.

Obs.: Para usar um decremento, ou seja, um valor que subtrai, apenas use um número negativo: -1, -2, -5, -10 ... etc.

Veja os exemplos:

📋
 ex2.prisma
imprima('---------- Início padrão: 1, incr padrão: 1 -----');
local cont = contador();//padrão inicia por 1, o incremento é 1;
imprima(cont());
imprima(cont());
imprima(cont());

imprima('----- Início: 2, incr padrão: 1 --------');
local cont2 = contador(2);//inicia por 2, o incremento é 1;
imprima(cont2());
imprima(cont2());
imprima(cont2());

imprima('------ Início: 10, incr: 5 -------');
local cont3 = contador(10, 5);
imprima(cont3());
imprima(cont3());
imprima(cont3());

imprima('-------- Início: 20, decr: -2 -------');
local cont4 = contador(20, -2);
imprima(cont4());
imprima(cont4());
imprima(cont4());

imprima('-------- Ótimo para gerar ids --------');
local contid = contador();
funcao crie_id()
    retorne string.formate('id_%04x', contid());
fim
imprima('<button id="'..crie_id()..'">Baixar</button>');
imprima('<button id="'..crie_id()..'">Enviar</button>');
imprima('<button id="'..crie_id()..'">Login</button>');
