funcao jpg.erro(s)

Função jpg.erro(s) - define ou retorna o último erro.

Parâmetros:

s {string ou nulo} - se string então define uma string como último erro e a retorna, se nulo retorna o último erro.

Retornos:

{string} - o retorno é uma string contento a descrição do último erro.
Voltar ao início

funcao jpg.info(arq)

Obtém as informações dentro de um arquivo JPG ;

Parâmetros:

arq(string) - nome do arquivo JPG a ser analisado.

Retornos:

(tabela) contendo os seguintes campos:

Campos da tabela retornada:

  • alt = altura da imagem em pixels
  • larg = largura da imagem em pixels
  • id = identificador da imagem JFIF
  • cs = canal de cor (DeviceRGB ou DeviceGray ou DeviceCMYK)
  • bpc = bits por componentes;
  • tam = tamanho do arquivo em bytes;
  • f = decoder para pdf: DCTDecode;
  • versao = versão do jpeg (1.01 ou 1.02)
  • versao_maior = 1
  • versao_menor = 1 ou 2

funcao jpg.info_str(a)

Obtém as informações de um JPG a partir de uma string.

Parâmetros:

txt_img(string) - o conteúdo da imagem lida em string \[use a função es.abra(arq_jpg, 'leiturabin')\]

Ao optar por este método, o programador fica responsável por ler a imagem de um arquivo para uma string e depois passar a string para esta função.

Retornos:

(tabela) contendo os seguintes campos:
  • alt = altura da imagem em pixels
  • larg = largura da imagem em pixels
  • id = identificador da imagem JFIF
  • cs = canal de cor (DeviceRGB ou DeviceGray ou DeviceCMYK)
  • bpc = bits por componentes;
  • tam = tamanho do arquivo em bytes;
  • f = decoder para pdf: DCTDecode;
  • versao = versão do jpeg (1.01 ou 1.02)
  • versao_maior = 1
  • versao_menor = 1 ou 2
