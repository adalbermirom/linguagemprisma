funcao t.cor.rgb(r, g, b)

Em terminais com suporte a '256color' esta função retorna um número da cor a partir da combinação rgb de 0-5.

Parâmetros:

r {numero} - um número de 0 a 5 para red (vermelho)
g {numero} - um número de 0 a 5 para green (verde)
b {numero} - um número de 0 a 5 para blue (azul)

Retornos:

ret {numero} - Um número correspondente ao código da cor 256.

Obs.: Esta função só pode ser usada em terminais com suporte a 256color. Para isso faça o teste: se t.SUPORTE_256COLOR entao ... fim

funcao t.escolha(...)

Cria um diálogo com opções para se escolher.

Parâmetros:

... {variado - strings} - primeiro: título, os demais são as opções.

Retornos:

op {numero} - O número da opção escolhida.
txt {string} - O rótulo da opção escolhida.

Exemplo:

📋
 prisma
 local tm = inclua'tmsg';

 local op, rot = tm.escolha('O que quer fazer?', 'Desligar', 'Reiniciar', 'Suspender');
 imprima(op, rot);// op = rot --> 1 = Desligar, 2 = Reiniciar, 3 = Suspender.

funcao t.login (titulo, txt_user, txt_pass)

Cria um diálogo de login, com usuário e senha.

Parâmetros:

titulo {string} - título do diálogo.
txt_user {string} - texto correspondente ao rótulo 'User', permitindo customização.
txt_pass {string} - texto correstondente ao rótulo 'Password'.

Retornos:

user {string} - O que foi digitado no usuário.
senha {string} - O que foi digitado na senha.

Exemplo:

📋
 prisma
local tm = inclua'tmsg';

local user, senha = tm.login('Faça o login', 'Usuário', 'Senha');
imprima(user, senha);

leia();

t.dialogo_cor = funcao(txt)

Cria um diálogo para selecionar uma cor.

Parâmetros:

txt {string} - A mensagem que aparece acima do diálogo.

Retornos:

cor {numero} - O código correspondente à cor escolhida.

Obs.: Para terminais sem suporte a vt100 e cores, esta função não terá efeito.

Obs.: Para terminais com suporte a 256color, há mais cores além das 16 básicas.

Variáveis

Voltar ao início

t.SUPORTE_256CORES = suporte_256cores();

{booleano} - se verdadeiro, há suporte a 256cores, caso contrário não.

Voltar ao início

t.SUPORTE_VT100 = suporte_vt100()

{booleano} - se verdadeiro apresenta suporte ao sistema de escape vt100.
