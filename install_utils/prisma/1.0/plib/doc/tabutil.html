função tab.convstring (t, c, i, ncol)

Converte uma tabela com todos os seus elementos para sua representação em string.

Parâmetros:

t(tabela) - a tabela para ser convertida em string.
c(string ou nulo) - um caractere ou string separador de cada elemento, geralmente um espaço' ' ou nova linha'\n'

Obs.: O parâmetro c(string) é opcional seu valor padrão é espaço ' ';

i(verdadeiro ou falso ou nulo) - se falso ou nulo, os indices serão adicionados, se verdadeiro ou qualquer outro valor válido, índices serão ignorados.

Obs.: O parâmetro i é opcional, seu valor padrão é 'nulo', ou seja, adiciona os índices da tabela na string final.

Obs.: Os campos sempre são adicionados!

ncol (numero ou nulo) [opcional] - define o número de colunas, insere um quebra linha '\n', o padrão é sem colunas.

Retornos:

(string) - o retorno é a string, a representação da tabela convertida.

Veja um exemplo:

📋
 ex1.prisma
local tab = inclua'tabutil';
local t = {'ola','tudo','bem', nome='Aldebaran', pos={fig="quad", x=1,y=2,alt=10}};
local s = tab.convstring(t);
imprima(s);
//saída: { [1] = "ola"; [2] = "tudo"; [3] = "bem"; nome = "Aldebaran"; pos = { y = 2; fig = "quad"; x = 1; alt = 10 } }

função tab.cada(t, f, começo, final)

Executa uma callback para cada elemento da tabela, passando o elemento como parâmetro.

Obs.: Funciona apenas com tabela indexadas (ignora campos se houver).

Parâmetros:

t (tabela) - é a tabela para percorrer todos os elementos.
f (funcao) - é a função para ser executada para cada elemento da tabela. 
protótipo de f -->   função f(el) onde el é o elemento da tabela;
começo(numero ou nulo) [opcional - padrão = 1] - é o número do índice inicial ao percorrer a tabela.
final(numero ou nulo) [opcional - padrão = #t] - é o número do índice final ao percorrer a tabela.

Retornos:

(nulo) - não há retorno.
📋
 Exemplo:
local tab = inclua'tabutil';
local t = {'dom','seg','ter','qua','qui','sex','sáb'};
tab.cada(t, imprima);

função tab.inc(t, f, começo, final, n, dado)

Obs.: Funciona apenas com tabela indexadas (ignora campos se houver).

Semelhante à função anterior mas com mais recursos, em vez de passar só o elemento para função callback, passa outros parâmetros, veja abaixo:

Parâmetros:

t (tabela) - é a tabela para percorrer todos os elementos.
f (funcao) - é a função para ser executada para cada elemento da tabela. 
 protótipo de f -->   função f(t, i, dado) onde t é a tabela, i é o índice e dado é o dado extra.
começo(numero ou nulo) [opcional - padrão = 1] - é o número do índice inicial ao percorrer a tabela.
final(numero ou nulo) [opcional - padrão = #t] - é o número do índice final ao percorrer a tabela.
n (numero ou nulo) [opcional - padrão = 1] - é o incremento ou decremento a ser usado no laço 'para'
dado(qualquer valor) [opcional - padrão = nulo] - é o dado extra a ser passado para callback.

Retornos:

(nulo) - não há retornos.
📋
 Exemplo:
local tab = inclua'tabutil';
funcao callback(t, i, dado)
    imprima(dado, t[i]);
fim

local t = {'dom','seg','ter','qua','qui','sex','sáb'};
tab.inc(t, função (t,i) imprima(t[i]); fim, 1, 3);
tab.inc(t, callback, 6, 1, -1, 'Dia da semana:');
//usamos o incremento -1 ou seja diminui 1 a cada execução da callback;

funcao tab.map(t, f, começo, final, dado)

Mapeia todos os índices da tabela, trocando seus valores pelos valores retornados da callback.

Obs.: Funciona somente com tabelas indexadas (array), campos são ignorados.

Parâmetros:

t (tabela) - é a tabela para percorrer todos os elementos.
f (funcao) - é a função para ser executada para cada elemento da tabela. 
 protótipo de f -->   função f(t, i, dado) onde t é a tabela, i é o índice e dado é o dado extra.
começo(numero ou nulo) [opcional - padrão = 1] - é o número do índice inicial ao percorrer a tabela.
final(numero ou nulo) [opcional - padrão = #t] - é o número do índice final ao percorrer a tabela.
dado(qualquer valor) [opcional - padrão = nulo] - é o dado extra a ser passado para callback.

Obs.: Note que esta função não tem o incremento como parâmetro, pois não é necessário para o que ela faz.

Retornos:

(nulo) - não há retornos.
📋
 Exemplo:
local tab = inclua'tabutil';
local t = {'dom','seg','ter','qua','qui','sex','sáb'};

tab.cada(t, imprima);

imprima('--------------------');

tab.map(t, funcao(t,i)
    retorne t[i]:maiuscula();
fim);

tab.cada(t, imprima);
