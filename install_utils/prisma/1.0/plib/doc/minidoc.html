Documentação - minidoc.pris

Simples bilioteca para gerar documentação a partir de fontes (scripts) Prisma.

*

Esta lib lê um arquivo Prisma em busca de marcações de documentação para gerar um arquivo html com toda a documentação extraída.

Especificações:

Comentários

A documentação é gerada a partir de marcações nos comentários de fontes da linguagem Prisma.

Todo comentário de documentação deve iniciar por /** ou por /*@ e ser finalizado por um * /

Tags

A marcação é feita por meio de tags que iniciam com o caractere \ ou @ e cada tag tem um significado e função para o minidoc.

Exemplos de tags:

\fonte md5.pris
\autor Adalberto
\versao 0.2
\data nov/21
\resumo Biblioteca para gera hashs md5.

A tag que não pode faltar: fonte (ou file)

Todo arquivo que se pretende gerar uma documentação com o minidoc precisa de uma tag fonte caso contrário o arquivo é descartado, mesmo tendo outras tags.

Veja o exemplo:

\fonte md5.pris

 

 

 

Estrutura

Cabeçalho (header)

O primeiro bloco de comentário deve conter algumas informações necessárias para o cabeçalho da documentção: nome do arquivo, autor, data, resumo, imagem e texto.

A tag fonte é indispensável, as outras podem faltar. Tudo o que estiver fora de uma tag seja de uma linha ou multinha será considerado um texto html, por isso podemos usar a codificação html a vontade nas explicações dos comentários.

Podemos usar ainda as tags de bloco e de código, para gerar uma saída html com exibição de códigos fontes Prisma.

Veja um exemplo de estrutura header completa:

/**
   \fonte meu_fonte.prisma
   \autor Nome do Autor
   \versao 0.2
   \data nov 2023
   \resumo Este programa serve como exemplo para esta documentação ... 

  <p>Escreva mais sobre o que desejar nas próximas linhas usando <b>código html</b></p>
  
  \inicio_codigo Exemplo:
    //entre essas tags você pode colocar comandos Prisma e ele será apresentado com destaque na documentação.
    imprima("apenas um exemplo!");
    para i = 1, 10 inicio
        imprima(i, 'olá mundo!');
    fim    
  \fim_codigo
 */
